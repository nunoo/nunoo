# nunoo

I’m the lead software engineer behind [Reliant](https://github.com/reliant-labs/reliant), an AI coding assistant built for real software development workflows, multi-agent execution, and programmable automation.

I spend most of my time designing developer tools, shaping agent systems, and building products that make complex engineering work feel fast, reliable, and composable.

---

## What I’m building

### [Reliant](https://github.com/reliant-labs/reliant)
Reliant is an AI coding assistant that goes beyond chat.

It combines:
- LLM-powered code understanding
- programmable workflows
- multi-agent execution
- MCP integrations
- git worktree-based development
- desktop and web tooling

The goal is simple: make software engineering agents actually useful in real repositories.

---

## Current focus

- building better agent workflows for software engineering
- making multi-workspace and multi-worktree development feel native
- improving developer tooling around automation, recovery, and orchestration
- pushing product UX for serious technical users
